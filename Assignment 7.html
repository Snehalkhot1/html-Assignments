3. Understanding Paths

In the code above, I used an Absolute Path (a full URL). If the image was on your computer, you would use a Relative Path:

Type Code Example
Absolute src="https://website.com/photo.jpg"
Relative (Same Folder) src="mountain.jpg"
Relative (Sub-folder) src="images/mountain.jpg"
